Prosta aplikacja koszyka produktów w React.
🚀 Jak uruchomić
1. Sklonuj repozytorium
git clone https://github.com/1LvlDeath1/react-cart
2. Przejdź do katalogu projektu
cd react-cart
3. Zainstaluj zależności
npm install
4. Uruchom projekt
npm start
